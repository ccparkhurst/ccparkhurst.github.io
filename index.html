PeakBagr

A Node.js, Express.js, and MongoDB app to keep track of all of the mountains you have climbed using Google Maps API

Peakbagger (n): An individual who participates in peak bagging, an activity in which hikers and mountaineers attempt to reach the summit of a mountain peak by any route using only human power.

PeakBagr is an app which allows users to keep track of all of the peaks they have bagged and those they wish to bag. A user can drop a pin on the mountain peak they wish to claim and add it to their list, and then view the location of all peaks in their collection on one handy map.

Xylophone Says

Front-end JavaScript browser game which tests a user’s memory skills

To play a game of Xylophone Says, the player clicks on the xylophone mallet to begin the game. The computer will flash a key and play a sound, and the player must match the computer by clicking on the same key. The computer will go again, this time adding one new key to the sequence before the player goes again. This continues until the player clicks on the wrong key, ending the game. Womp womp.

Built With:

  • JavaScript
  • jQuery
  • HTML5
